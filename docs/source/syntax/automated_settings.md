Elastic Docs V3 supports the ability to build a markdown settings reference from a YAML source file.

### Syntax

:::none
\`\`\`\{settings} kibana-alerting-action-settings.yml
\`\`\`
:::

### Result

_Everything below this line is auto-generated._

