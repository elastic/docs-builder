## Usage examples

You can use icons in various Markdown elements.

### Headings

You can include icons in headings:

::::{tab-set}

:::{tab-item} Output

#### Test the :gear: Settings
#### User Profile :user:

:::

:::{tab-item} Markdown
```markdown
#### Test the :gear: Settings
#### User Profile :user:
```
:::

::::

### Lists

Icons can be used in both ordered and unordered lists.


::::{tab-set}

:::{tab-item} Output

**Unordered List:**
* :check: Task one completed
* :check: Task two completed
* :cross: Task three pending

**Ordered List:**
1. :check: First item
2. :check: Second item
3. :cross: Third item

:::

:::{tab-item} Markdown
```markdown
**Unordered List:**
* :check: Task one completed
* :check: Task two completed
* :cross: Task three pending

**Ordered List:**
1. :check: First item
2. :check: Second item
3. :cross: Third item
```
:::

::::

### Tables

Icons can be used within table cells.

::::{tab-set}

:::{tab-item} Output

|     Status     | Description |
|:--------------:|:------------|
| :checkCircle: | Success     |
|   :warning:   | Warning     |
|    :error:    | Error       |

:::

:::{tab-item} Markdown
```markdown
|     Status     | Description |
|:--------------:|:------------|
| :checkCircle: | Success     |
|   :warning:   | Warning     |
|    :error:    | Error       |
```
:::

:::{tab-item} Markdown
