# assembler serve

Serve the output of an assembler build on `http://localhost:4000/`

## Usage

```
docs-builder assembler serve [options...] [-h|--help] [--version]
```

## Options

`--port` `<int>`
:   Port to serve the documentation. (Default:   4000)

`--path` `<string>`
: (optional)
