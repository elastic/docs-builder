Run a search

Server: https://fixture.example.com (Fixture server)

Paths /{index}/_search

  • POST /{index}/_search

Prerequisites /{index}/_search

  • Requires the `read` index privilege for the target data stream or index.

  • Requires the `monitor` cluster privilege when profiling is enabled.

Path Parameters

index

A comma-separated list of index names to search.

Description /{index}/_search

Returns hits that match the query defined in the request. Supports pagination through `size` and `from`.

Read the reference documentation
Authorization: Authorization

Request Examples /{index}/_search

Boolean query

  }
