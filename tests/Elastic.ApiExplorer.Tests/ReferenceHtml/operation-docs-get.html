Get a document by id

Server: https://fixture.example.com (Fixture server)

Paths /{index}/_doc/{id}

  • GET /{index}/_doc/{id}
  • GET /{index}/_source/{id} deprecated

Path Parameters

index

The name of the index that contains the document.

id

The unique document id.

Description /{index}/_doc/{id}

Retrieves the document with the specified id from the index.

Response application/json /{index}/_doc/{id}

Response Type: [] DocumentHit

_id string optional
_score number optional
_source object optional
